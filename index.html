To obtain the project from the repository, write into your console the command:

git clone git@github.com/jbonnet44300/Datatypes.git

To access to results of unit testing, go to the directory where you have clone the repository and write into your console:

mvn compile

mvn test

To access to the report code analysis (PMD), write into your console:

mvn pmd:pmd -DtargetJdk=1.6 and go to > directory/target/site/pmd.html
